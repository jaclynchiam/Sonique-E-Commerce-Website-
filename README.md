# Sonique-E_Commerce_Website
Sonique is a front-end e-commerce website designed using Figma and developed with HTML, CSS, and JavaScript as part of a Web Fundamentals assignment in May 2025. It simulates a modern online store for headphones and earphones, featuring a clean UI, responsive layout, and interactive user interface components.
